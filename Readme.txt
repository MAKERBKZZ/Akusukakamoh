╠════════════════════════════════════════════════════════════════╣
║  🛠 Deskripsi:                                                 ║
║  Script ini adalah alat otomatisasi untuk layanan sosial      ║
║  media seperti TikTok, Instagram, dan Twitter. Script ini     ║
║  dirancang untuk dijalankan di Windows maupun Android (Termux)║
║  dengan antarmuka elegan dan fitur verifikasi lisensi         ║
║  berbasis online.                                             ║
╠════════════════════════════════════════════════════════════════╣
║  ✅ Fitur:                                                     ║
║  - Verifikasi lisensi berbasis waktu dan IP                   ║
║  - Antarmuka pengguna elegan menggunakan Rich dan Colorama    ║
║  - Kompatibel di Termux dan Windows                           ║
║  - Sistem logging pengguna terintegrasi                       ║
║  - Otomatisasi request berbasis cloudscraper (bypass anti-bot)║
╠════════════════════════════════════════════════════════════════╣
║  📦 Cara Instalasi:                                           ║
║  1. Pastikan Python 3.x telah terinstal                       ║
║  2. Install Modul dengan:                                     ║
║     pip install -r requirements.txt                           ║
║  3. Jalankan script dengan:                                   ║
║     python sosmed.py                                           ║
